As the rand() function relies on a pseudorandom number generator, it should not be used for security-critical applications or for protecting sensitive data. In such context, the methods random_int() or random_bytes() or openssl_random_pseudo_bytes() which relies on a cryptographically strong random number generator (RNG) should be used in place.

Noncompliant Code Example

$random = rand();

Compliant Solution

$randomInt = random_int(); // Compliant; generates a cryptographically secure pseudo-random integer
